## Description

This is the sample package of nestjs microservice hybride module created by Chetan Sharma.
First you need to install dependencies for each of services then run each service in a new server or port.

### List of API's :
(1) Call microservice 1 : http://localhost:3000/microservice1
(2) Call microservice 2 : http://localhost:3000/microservice2

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
